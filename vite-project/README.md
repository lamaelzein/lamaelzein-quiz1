React + TypeScript + Vite Project
📌 Description

This is a modern React application built using Vite and TypeScript. The project demonstrates fast development setup, component-based architecture, and type-safe coding practices for building scalable web applications.

🚀 Setup Instructions
1. Install dependencies
npm install
2. Run the development server
npm run dev
3. Build for production
npm run build
4. Preview production build
npm run preview
